# Frontend Yuki (React + Vite)

## Installation

```bash
cd frontend
cp .env.example .env
npm install
npm run dev
```

L’interface est sur http://localhost:5173 — l’API doit tourner sur http://127.0.0.1:8000 (voir `backend/README.md`).

## Scripts

- `npm run dev` — serveur de développement
- `npm run build` — build de production
- `npm run lint` — ESLint
